## ⚠️ BREAKING CHANGE: Library-Specific Tools Removed

This is a **semver MAJOR version bump** (v1.3.0) due to breaking API changes.

### What's Removed

The following MCP tools are **permanently removed**:
- `search_rtl` — no longer available
- `lookup_rtl_class` — no longer available
- `search_devexpress` — no longer available
- `lookup_devexpress_class` — no longer available

**Result**: Attempting to call these tools will result in a "tool not found" error.

### What to Do

If you're upgrading from v1.2.0 or earlier, you **must** migrate to unified tools before this upgrade:

**v1.2 code** (no longer works):
```csharp
var result = await mcp.CallTool("search_rtl", new { query = "TStringList", topK = 5 });
```

**v1.3 code** (required):
```csharp
var result = await mcp.CallTool("search_delphi_source", new { 
    query = "TStringList", 
    library = "rtl",
    topK = 5 
});
```

### Full Migration Guide

| v1.2 Tool | v1.3 Replacement | Notes |
|---|---|---|
| `search_rtl(query, topK, version)` | `search_delphi_source(query, library="rtl", topK, version)` | Add explicit `library` parameter |
| `lookup_rtl_class(className, version)` | `lookup_delphi_class(className, library="rtl", version)` | Add explicit `library` parameter |
| `search_devexpress(query, topK, version)` | `search_delphi_source(query, library="devexpress", topK, version)` | Add explicit `library` parameter |
| `lookup_devexpress_class(className, version)` | `lookup_delphi_class(className, library="devexpress", version)` | Add explicit `library` parameter |

### With Profile Defaults (Recommended)

Configure your profile with `DefaultScopes` to simplify queries:

```json
{
  "ClientAccess": {
    "Profiles": {
      "my-machine": {
        "DefaultScopes": ["rtl", "devexpress"],
        "MaxTopK": 10
      }
    }
  }
}
```

Then omit the `library` parameter for multi-library searches:

```csharp
// Profile's DefaultScopes will be used automatically
var result = await mcp.CallTool("search_delphi_source", new { 
    query = "TStringList",
    topK = 5  // searches RTL + DevExpress per profile
});
```
