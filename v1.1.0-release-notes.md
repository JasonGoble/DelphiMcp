## ✨ What's New

### Unified Delphi Source Tools
- **New tools**: `search_delphi_source`, `lookup_delphi_class`
- **Replaces**: 4 library-specific tools (search_rtl, lookup_rtl_class, search_devexpress, lookup_devexpress_class)
- **Benefit**: Simpler API, automatic scope resolution via machine profiles

### Machine Profile Access Control (Issue #26)
- Per-API-key machine profile configuration
- Global and per-profile policy enforcement:
  - `MaxTopK`: Limit search results
  - `MaxVersionsPerLibraryPerQuery`: Limit multi-version class comparisons
- HTTP auth middleware with automatic profile resolution

### Multi-Version Class Lookup
- Compare classes across multiple versions in a single query
- Configurable limit via `MaxVersionsPerLibraryPerQuery` policy
- Useful for tracking API changes across Delphi/DevExpress releases

### Automatic Scope Resolution
- Specify library in tool call → use that library
- Omit library → use profile's `DefaultScopes` (e.g., [RTL, DevExpress])
- Fallback to common libraries (RTL + DevExpress) in stdio mode

## 📋 Migration Guide

See **README.md: v1.0 → v1.1 Migration Guide** for detailed upgrade steps.

### Quick Start: Upgrading from v1.0

**Before (v1.0)**:
```
Tool: search_rtl
Params: term: 'String', topK: 10
→ Search only RTL
```

**After (v1.1)**:
```
Tool: search_delphi_source
Params: term: 'String', library: 'RTL', topK: 10
→ Search RTL (explicit) or auto-scoped via profile
```

### Configuration Migration

**Legacy (v1.0)**:
```json
{ "Hosted": { "ApiKey": "abc123" } }
```

**New (v1.1)**:
```json
{
  "ClientAccess": {
    "Profiles": {
      "default": {
        "DefaultScopes": ["RTL", "DevExpress"],
        "MaxTopK": 50,
        "MaxVersionsPerLibraryPerQuery": 3
      }
    },
    "ApiKeyToProfile": { "abc123": "default" }
  }
}
```

## ⚠️ Deprecation Notice

The following tools are **deprecated as of v1.1** and will be **removed in v1.2**:
- `search_rtl` → migrate to `search_delphi_source` with `library: 'RTL'`
- `lookup_rtl_class` → migrate to `lookup_delphi_class` with `library: 'RTL'`
- `search_devexpress` → migrate to `search_delphi_source` with `library: 'DevExpress'`
- `lookup_devexpress_class` → migrate to `lookup_delphi_class` with `library: 'DevExpress'`

**Migration Timeline**:
- **v1.1** (now): New unified tools available; old tools functional but deprecated
- **v1.2** (next release): Old tools removed; profiles required
- **v1.1 users**: Upgrade at your own pace; mixed use of old/new tools supported during transition
