# Architecture Decisions

This folder contains Architecture Decision Records (ADRs) for the DelphiMcp project. Each ADR documents a key design or workflow decision, the context and rationale, and any consequences for the codebase or workflow.

- ADRs are numbered sequentially (e.g., 0001, 0002, ...).
- Each ADR is referenced from the main project README for discoverability.
- Use this folder to track major technical, architectural, or workflow choices that affect the project over time.
